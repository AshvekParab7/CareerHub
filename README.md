🤖 CareerHub – AI Career Guidance Platform

 🚀 Overview
CareerHub is an AI-powered career guidance platform that recommends suitable career paths based on user responses and interests.

The system also provides curated study materials, structured learning roadmaps, and an integrated resume builder.


 ✨ Features
- AI-based career recommendation system
- Interactive user questionnaire
- Career bookmarking and comparison
- Study material resources by career field
- AI-generated learning roadmap
- Built-in resume builder
